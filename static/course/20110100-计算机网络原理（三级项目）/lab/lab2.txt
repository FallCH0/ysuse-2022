一、 实验目的

        1. 了解常用网络设备及相关功能特性。

        2. 学会使用思科路由模拟器，利用路由模拟器掌握两层及三层交换机配置的方法。

        3. 了解vlan的配置和作用，了解网络基本配置原理。

        4. 具备一定的自学能力。

二、 实验环境

        1. 硬件环境：计算机一台。

        2. 软件环境：思科路由模拟器（Cisco Packet Tracer）模拟出pc若干；两层及三层交换机，网线若干。

三、 实验步骤

        根据实验步骤内容，学会如何配置交换机，不用写入实验报告。

        1. 了解思科路由模拟器的使用方法，掌握两层及三层交换机的基本配置方法。

        2. 利用思科模拟器，搭建由思科2950交换机和若干台pc组成的基本网络，学习交换机的基本配置方法。

              1） 模拟器中添加一台思科2950交换机，配置交换机名为sw1，管理地址为192.168.0.2，网关指向192.168.0.1，掩码255.255.255.0。添加两台计算机pc1和pc2，并将两台pc用直连线连接到sw1的端口：pc1-端口1，pc2-端口2。

                2） 配置sw1的端口1和2分别处于vlan 2和vlan 3中。（思考：此时交换机上有几个vlan）

                3） 用网线连接pc1和pc2到这两个端口，在pc上分别配置相应的ip地址，pc1：192.168.1.2，pc2：192.168.1.3。

                4） 在每台pc上ping自己的IP，观察现象。

                5） 然后再ping对方地址，观察现象并分析。

                6） 将端口2改到vlan2，再ping另外一台pc，观察并分析。将端口2shutdown，再用ping检测网络，观察并分析。

                7)  增加一台交换机，配置名称为sw2，管理地址为192.168.0.3，配置网关指向192.168.0.1。sw2和sw1用交叉线连接最后一个端口，并把最后一个端口配置成级联模式（trunk），利用交换机控制台检测交换机网络是否通畅。

                8）在sw2上增加pc5，接入vlan2，配置ip地址为192.168.1.4，利用网络命令检测和sw1上的pc的通讯情况并分析。

        3. 搭建由一台思科3560三层交换机和二台pc组成的基本网络，学习三层交换机的基本配置方法。

                1） 在2的基础上添加一台3560交换机，在交换机上配置vlan 2和vlan 3，接口地址分别为192.168.1.1/24和192.168.2.1/24。开启三层路由模式（学会三层网络的配置方法）

                2） 配置交换机的端口1和2分别处于vlan 2和vlan 3中，用网线连接pc到这两个端口，在pc上分别配置相应的ip地址，pc1：192.168.1.5，pc2：192.168.2.2。（要懂得终端网络配置信息和三层配置之间的关系）

                3） 然后在每台pc上ping自己的网关，观察现象，再ping对方地址，观察现象并分析。

                4）连接三层交换机和sw1，配置好连接端口，测试pc之间的通信情况。

附件： 常用交换机配置命令

1. Enable 进入特权模式，在交换机初始化状态后，提示符为>，此时无法使用配置命令，需要进入特权模式才能配置交换机。

2. Conf  t  全局模式命令，此命令进入全局配置模式，可以在此模式下进行全局模式的配置，如建立vlan 、开启路由模式等。

3. Int fa0/1 进入fa0/1子接口，fa0/1指的是百兆交换机的第一个端口。

4. Switch access vlan 2  此命令为子接口命令，如在fa0/1下可配置此命令，意义为将第一个端口配置到vlan2中，此时端口1下连接的计算机使用的地址段就是vlan 2的地址段。

5. 建立并配置vlan 1

Int  vlan 1

Ip   address 192.168.1.1 255.255.255.0

No  shutdown

6. 开启路由，在全局模式下运行ip  routing命令。

7. Ospf路由配置，进入ospf路由配置router ospf 1，配置路由项

network 192.168.3.0 255.255.255.0

8. 保存配置  修改完交换机或者路由器的配置后，需要保存，否则交换机关机后，新配置会丢失。命令：   write  或 wr

9. 查看正在运行的配置    命令：sh run

    查看三层交换机或者路由器的路由表：sh ip route

    查看mac表：sh mac address-table

    查看arp列表：sh arp
